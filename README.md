### **Scrap Runners**
Theme / Setting / Genre
**Physics-Based Arena Shooter:** set in a decaying industrial wasteland where autonomous bot-rigs battle for dwindling resources. The game features high-speed scrap salvage, strategic point delivery, and momentum-driven kinetic combat, with a gritty, high-stakes mechanical tone.

### **Description**
Scrap Runners are fast-paced, player-controlled bots built for short, intense matches. Each bot wields up to two weapons at a time and competes to collect scrap scattered throughout the arena, delivering it to score points. Matches last ten minutes, and the winner is the player with the highest score at the end. The game emphasizes quick decision-making, dynamic movement, and tactical use of weapons, creating a tight, satisfying gameplay loop centered around collecting, delivering, and outmaneuvering opponents.
